OpenBD新刊フィード作成サービス

OpenBDから新刊情報を取得するRSSフィードを作成できます。条件を入力して、あなた専用のフィードを作成しましょう。

使用方法

  • フィード名と検索条件を入力してフィードを作成
  • 生成されたURLをRSSリーダーに登録
  • 毎日午前4時に自動更新
  • 条件に一致する新刊が見つかるとフィードに追加
